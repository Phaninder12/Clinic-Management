# Clinic Management System

A web application for managing clinic operations, featuring patient management, appointment scheduling, billing, and a secure login system built with JavaScript.

## Features

- **JavaScript Login System:** Secure authentication for staff and administrators using JavaScript. Only authorized users can access management features.
- **Patient Management:** Add, edit, and view patient records.
- **Appointment Scheduling:** Book, update, and cancel appointments.
- **Billing:** Generate and manage patient bills.
- **Dashboard:** Overview of clinic activities and statistics.
- **Admin Panel:** Manage users and clinic settings.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Authentication:** JavaScript-based login (session/cookie/localStorage)

## Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, etc.)

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/Phaninder12/clinic-management.git
   cd clinic-management
   ```

2. **Run the application:**
   ```sh
   npm start
   ```
   Or open `index.html` directly in your browser for frontend-only demo.
